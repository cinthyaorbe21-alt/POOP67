# Proyectos POO P67 

En esta presentación se muestran los proyectos desarrollados en la materia de Programación Orientada a Objetos. Aquí se explica cómo se aplicaron conceptos como clases, objetos y herencia para crear soluciones prácticas y organizadas. El objetivo es demostrar lo aprendido y cómo fue llevado a la práctica a través de los programas realizados.







# Unidad 1

### 1.1.- Fundamentos de Raspberry Pi y Python
En esta sección se comprende como una computadora pequeña puede interactuar directamente con componentes electrónicos como un LED y un Botón.

##### *Aprendizajes:*

- Aprender a estructurar un código que interactúa con hardware en tiempo real.
- Conectarse vía SSH a la Raspberry Pi3 desde otra computadora.
- Ejecutar programas sin necesidad de un monitor y teclado concetados directamente.
 
### 1.2.- Control de LED y Botón con Funciones
En lugar de escribir todo el código en un solo bloque, se usan funciones para organizar el programa, manteniendo el programa más ordenado y fácil de entender.

##### *Aprendizajes:*

- Aprender a crear módulos para separar secciones en funciones.
- Control interactivo con los componentes electrónicos como un LED y un Botón.

### 1.3.- POO con Herencia y Módulos
En esta práctica enseña a pensar en objetos y módulos para controlar hardware, mostrando cómo la programación orientada a objetos hace que el código sea más ordenado.

##### *Aprendizajes:*
- Se entendió que la Raspberry Pi no solo ejecuta scripts simples, sino que puede manejar arquitecturas más complejas.

# Unidad 2

### 2.1.- Arquitectura MVC en Raspberry Pi
Aplicar MVC en Raspberry Pi 3 con PuTTY nos enseña a estructurar proyectos de hardware y software de forma ordenada.

##### *Aprendizajes:*
- modelo: sabe como funciona el hardware.
- vista: se encarga de mostrar resultados.
- controlador: conecta modelo y vista y define el flujo.

### 2.2.- Control de Raspberry Pi con Telegram
El control con Telegram convierte a la Raspberry Pi en un sistema que puedes manejar a distancia desde tu celular, enviando simples mensajes al bot para interactuar con el hardware.

##### *Aprendizajes:*
- ejecutar y supervisar programas desde otra computadora vía SSH.
- aprender a controlar hardware a distancia mediante bots y comandos simples.

### 2.3.- Manejo de Archivos en POO
Cuando se trabaja con POO en Python, además de controlar hardware con LEDs, botones,etc. Se puede guardar y leer información en archivos. Esto es útil para registrar eventos, estados o mensajes del sistema.

##### *Aprendizajes:*
- combinar control de hardware con almacenamiento de datos.
